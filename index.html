Use Hexo to Create a Personal Blog on Github

What is Hexo?

Hexo is a fast, simple and powerful blog framework.

Installation

Install Git

Install Node.js

The best way to install Node.js is with Node Version Manager, nvm for short.

  • Install nvm

    • Install script

    • Manual install

Often I also put in a line to use a specific version of node. Add nvm use 0.12 to your ~/.bashrc, ~/.profile file.

  • Install Node.js

Must issue the command below to tell nvm to use the specific version of Node.js just installed.

Install Hexo

Create a repo on github named jasonkoo.github.io

On local machine
mkdir jasonkoo.github.io
cd jasonkoo.github.io
echo “# jasonkoo.github.io” >> README.md
git init
git add README.md
git commit -m “first commit”
git remote add origin git@github.com:jasonkoo/jasonkoo.github.io.git
git push -u origin master

hexo init .
hexo install

install theme
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

config theme
vi theme: yilia
theme: yilia
